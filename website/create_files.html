功能概述

批量创建文件功能是本工具的核心功能之一，适用于需要同时创建大量文件的场景。无论是创建测试数据、准备项目文件结构，还是生成报告模板，都能轻松应对。

使用教程

1. 直接输入方式创建文件

适合创建少量文件或文件名较简单的情况

  1. 选择"批量创建文件"功能
  2. 在左侧文本框中输入文件名列表，每行一个
  3. 指定目标文件夹路径
  4. 设置文件扩展名（如.txt、.docx等）
  5. 点击"创建文件"按钮执行

2. 从文件导入创建文件

适合批量创建大量文件或从已有数据生成文件

  1. 切换到"从文件导入"选项卡
  2. 点击"浏览"选择包含文件名的CSV、TXT或Excel文件
  3. 选择适当的列和行范围
  4. 指定目标文件夹和文件扩展名
  5. 点击"创建文件"按钮执行

3. 使用自定义命名规则

适合需要规范化命名或批量定制文件名的场景

  1. 点击"高级选项"展开更多设置
  2. 在"命名规则"中设置格式，如"报告_$NAME_$DATE"
  3. 可使用的变量包括：
    • $NAME - 原始文件名
    • $ISEQ - 自增序号
    • $DATE - 当前日期
    • $TIME - 当前时间
  4. 在"文件内容模板"区域设置文件内容
  5. 模板中可使用与命名规则相同的变量

使用技巧

  • 批量创建项目文件：可用于快速创建项目结构，如为多个客户创建相同结构的文档
  • 生成测试数据：创建大量测试文件时，利用自增序号变量生成连续编号的文件
  • 文档模板化：利用内容模板功能，为不同对象创建结构相同但内容不同的文档
  • 结合Excel使用：在Excel中整理好数据后导入，可以批量创建具有关联性的文件
